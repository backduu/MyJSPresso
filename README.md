# MyJSPresso
JSP로 만든 레거시 시스템이지만 여전히 진한 Espresso같은 프로젝트

# 사용기술
Front-end    : JSP + JAVASCRIPT + jQuery
Back-end     : Spring framework + MyBatis 
Database     : Oracle
Module       : Spring Web MVC
IDE & Server : IntelliJ IDEA Ultimate + tomcat 11
